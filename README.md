## 登录界面

### 概述

注册：账号通过邮箱进行注册，注册时输入自己的昵称和密码，若昵称有重复，则无法注册，提示更换昵称

登录：使用邮箱进行登录

找回密码：使用邮箱找回密码，并且重新设置密码

第三方登录：待定

### 方法

| 描述        | 返回类型         |
| ------------- | ------------ |
| 账号密码登录，验证是否匹配       | Boolean |
| 注册账号，验证昵称是否重复       | Boolean |
| 注册账号，将账号添加入数据库       | NULL |
| 重新设置密码       | NULL |
